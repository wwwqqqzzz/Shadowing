# Shadowing 影子跟读

> 听一句 → 跟一句 → 下一句。最小化从打开到开口的时间。

## 一句话介绍

Shadowing 是一个微信小程序 + NestJS 后端的英语跟读训练系统。核心闭环：播放一句原声 → 自动暂停 → 用户跟读 → ASR 评分 → 下一句。

## 技术栈

| 层 | 技术 |
|---|------|
| 小程序 | 原生 WXML+WXSS+JS |
| 后端 | NestJS + TypeORM + PostgreSQL |
| 管理后台 | React 19 + Ant Design 6 + Vite 8 |
| ASR | OpenAI Whisper (base, CPU) |
| 深色主题 | #0f0f0f 背景 / #a8e6cf 强调色 |

## 快速开始

```bash
# 后端
cd backend && npm install && npm run start:dev

# 管理后台
cd admin && npm install && npm run dev

# 小程序 — 微信开发者工具打开 miniprogram/ 目录
```

需要 PostgreSQL 数据库 `shadowing_dev`，TypeORM 自动同步表结构。

## 数据概况

- 13 个素材
- 1301 句子（100% en→zh-CN 翻译）
- 3 种跟读模式（自由/自动录音/手动）
- 错题本 + 连续2次≥80移出机制
- 打卡连续天数 + 3级日历视图
- 收藏功能
- 续练（从上次位置继续）
